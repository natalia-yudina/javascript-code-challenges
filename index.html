# 01_02 Movie Object

## What You'll Do

Create a Movie object that takes in five arguments:

  • title
  • director
  • genre
  • releaseYear
  • rating

    • Add a function called `getOverview` on the Movie prototype which console logs the following overview for each film:

    "<movie>, a <genre> film directed by <director> was released in <releaseYear>. It received a rating of <rating>."

## Requirements
  • Use object prototypes
